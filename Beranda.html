- Deskirpsi game

Game yang bernama TEKATI atau disebut "Tebak Kata Teknologi Informasi" merupakan game tebak kata yang mengusung tema teknologi informasi, jadi pertanyaan dan jawaban dalam game tersebut akan berisi semua tentang teknologi informasi/sistem informasi, dalam game tersebut mempunyai beberapa tingkatan, dalam 1 tingkat mempunyai 5-10 pertanyaan yang harus dijawab dengan benar agar bisa sampai ke stage selanjutnya, pemain akan diberikan kesempatan 3 kali, nyawa tersebut digunakan untuk melanjutkan menjawab soal apabila jawaban sebelumnya salah.

- Alasan membuat game tersebut

Alasan kami membuat game ini adalah untuk mengasah pengetahuan pemain dalam bidang teknologi informasi sehingga dapat dijadikan latihan sejauh mana kita mengetahui materi dalam bidang tersebut.

- Skenario atau Alur Cerita Game

Berikut adalah skenario atau alur dari game yang akan kami buat:

1. Pemain akan memasuki interface utama game.

2. Pada interface tersebut berisi button play untuk memainkan game, setting untuk mengakses pengaturan, dan exit untuk keluar dari game.

3. Jika pemain menekan tombol play, maka pemain akan diberikan pilihan tingkat/level kesulitan untuk memainkan nya, disini kami memberi 3 level, yaitu mudah, sedang, sulit.

4. Jika pemain memilih level mudah, maka pemain mendapatkan jatah mendapatkan clue sebanyak 5 kali dan memiliki rasio waktu paling lama.

5. Jika pemain memilih level sedang, pemain hanya mendapatkan jatah clue sebanyak 3 kali dan memiliki rasio waktu yang lebih pendek daripada level mudah.

6. Jika pemain memilih level sulit, maka pemain tidak mendapatkan jatah clue sama sekali dan mendapatkan rasio waktu paling sedikit.

7. Setelah pemain memilih level, maka pemain akan masuk kedalam permainan dan bisa menyelesaikan teka teki yang telah kami buat.

8. Setiap level mempunyai 10 soal yang wajib dijawab dengan benar oleh pemain.

9. Pemain akan diberi 3 nyawa yang berfungsi apabila jawaban pemain salah, maka pemain dapat melanjutkan game dengan menggunakan nyawa tersebut.
